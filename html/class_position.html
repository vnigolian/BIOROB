Detailed Description

Author
Valentin NIGOLIAN valen.nosp@m.tin..nosp@m.nigol.nosp@m.ian@.nosp@m.epfl..nosp@m.ch Fall 2015 This class is a simple triplet of integers that represent a discrete Position. It allows easier handling of grid-bound objects. The grid is also Roombot-sized. It has basic operators and methods, as expected from such object
