Detailed Description

The Scene contains all the Models used to represent the scene in which the simulation takes place. The floor, the walls, the windows, the roof, the door and a skybox surrounding the room Aside from the roof, all Models are added polymorphicly. The roof get a special treatment because it is not necessarily drawn, depending on the current mode.
